About tool

SVG to CSS Encoder

Modern web standards allow the inclusion of SVG image files directly within CSS via a data URI but requires encoding the SVG content, similar to base64 encoding binary images. SVG images encoded using Javascript's encodeURIComponent() built-in function allow the browser to properly read and render the image. This tool helps you do that.

Paste your SVG file and it will be encoded and provided as just the encoded image as well as the complete CSS background text needed to use. A preview of the encoded image is also provided to ensure nothing gets broken in the encoding process.
